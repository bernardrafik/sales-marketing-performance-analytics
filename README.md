# sales-marketing-performance-analytics
End-to-end BI project analyzing sales and marketing performance using Python, SQL and Power BI. 
- Data source: e-commerce (phone accessories)

## Tech Stack
- Python (data merging)
- Excel (data cleaning)
- SQL (data modeling & KPI calculations)
- Power BI (dashboard & visualization)

 ## Business aspects
- Revenue by country
- Marketing cost structure
- ROI / Ratio analysis
- Country performance comparison

## Project Structure
- data/ – raw sales & sale and marketing data, before and after merging and cleaning
- python/ – data merging script   
- sql/ – data modeling & KPI calculations  
- powerbi/ – final dashboard file
